Para ejecutar el programa:

* Necesita la carpeta node_module, por lo que se debe crear.
* El archivo que se debe ejecutar para iniciar el programa es principal.js,
	por lo que las instrucciones para ejecutar el programa son:
		* node principal // esta mostrará la lista de todos los cursos disponibles
		* node principal inscribir --id_cur='Id del curso' --nom_est='Nombre del estudiante a inscribir' --ced_est='Cédula del estudiante'
			// La anterior funciona como se especifica en la historia de usuario del entregable
